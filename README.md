# Александр — клинический психолог (Telegram бот)

Профессиональная психологическая поддержка с техниками КПТ, ACT и кризисной помощью.

## Команды
- /start — начать сессию
- /reset — сбросить историю
- /help — справка
- /grounding — техники заземления

## Особенности
- Эмпатичный, профессиональный стиль общения
- Реагирует на имена: Александр, Саша
- Техники 5-4-3-2-1, квадратное дыхание, сканирование тела
- Поддержка в групповых чатах (только при упоминании)
